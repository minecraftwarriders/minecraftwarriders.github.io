Core server commands and essential functionality

üìã Overview

EssentialsX provides the core functionality for Minecraft War Riders, including teleportation, economy, homes, warps, and essential server commands. It's the foundation that makes most other server features possible.

Most Important Plugin: EssentialsX commands are used constantly by every player. Master these commands and you'll have a much better server experience!

üåê Teleportation Commands

Move around the server quickly and efficiently:

/spawn
Teleport to the server spawn point
/tpa [player]
Request to teleport to another player
/tpahere [player]
Request a player to teleport to you
/tpaccept
Accept a teleport request
/tpdeny
Deny a teleport request
/back
Return to your last location (after death/teleport)
/top
Teleport to the highest block at your location
üí° Pro Tip: Use /back after dying to return to your death location and collect your items!

üè° Home Commands

üìç Warp Commands

üí° Home Organization: Use descriptive names for your homes like "mine", "farm", "base", "shop" to easily remember what each location is for!

üí∞ Economy Commands

Manage your money and trade with other players:

/balance
Check your current money balance
/bal [player]
Check another player's balance (if public)
/pay [player] [amount]
Send money to another player
/baltop
See the richest players on the server
/worth [item]
Check the server buy/sell value of an item
‚ö†Ô∏è Money Safety: Be careful when using /pay - payments cannot be reversed! Double-check the player name and amount before confirming.

üí¨ Communication Commands

Chat and communicate with other players:

/msg [player] [message]
Send a private message to a player
/r [message]
Reply to the last private message you received
/mail send [player] [message]
Send mail to a player (they'll receive it when they log in)
/mail read
Read your received mail messages
/mail clear
Delete all your mail messages
/ignore [player]
Block messages from a specific player
/unignore [player]
Stop ignoring a player
üí° Mail System: Use mail to communicate with players who aren't online. It's perfect for leaving messages about trades, builds, or just saying hello!
