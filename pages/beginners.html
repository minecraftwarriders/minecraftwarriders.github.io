üöÄ Getting Started

Welcome to Minecraft War Riders! This server has many custom features and plugins to enhance your gameplay experience. Here's what you need to know:

First Time Joining?
When you first join the server, you'll start with basic permissions and $1000 in your account. Use /spawn to teleport to the main hub area.
Join the Server: Connect using the server IP and wait for the world to load.
Read the Rules: Type /rules to see server rules and guidelines.
Set Your Home: Find a good location and type /sethome to set your home base.
Start Playing: Begin mining, building, and exploring to level up your skills!

üí∞ Economy System

The server has a comprehensive economy system with multiple ways to earn and spend money:

üí∏ Earning Money

  • Selling Items: Use /shop to sell items to the server
  • AutoSell Chests: Place items in AutoSell chests for automatic selling
  • Player Trading: Trade with other players using /trade
  • Auction House: Sell items to other players via /ah
  • Skills & Jobs: Level up skills to earn money from activities
üí° Pro Tip: AutoSell chests are perfect for farms! They automatically sell items when you place them inside, making farming very profitable.

üè™ Spending Money
