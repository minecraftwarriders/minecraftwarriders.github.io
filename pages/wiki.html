üéØ BEGINNER'S GUIDE üéØ

Everything you need to know to get started

üöÄ Getting Started

Welcome to Minecraft War Riders! This server has many custom features and plugins to enhance your gameplay experience.

First Time Joining?
When you first join the server, you'll start with basic permissions and $1000 in your account. Use /spawn to teleport to the main hub area.
Join the Server: Connect using the server IP and wait for the world to load.
Read the Rules: Type /rules to see server rules and guidelines.
Set Your Home: Find a good location and type /sethome to set your home base.
Start Playing: Begin mining, building, and exploring to level up your skills!

üì¶ AUTOSELLCHESTS üì¶

Automated selling system for farms and resource gathering

üìã Overview

AutoSellChests are special chests that automatically sell any items you put inside them. Perfect for farms and mining!

üõí How to Get: Buy from the server shop using /shop (EconomyShopGUI) but CANNOT be sold back. One-time investment!

üîß Usage

Purchase: Buy an AutoSellChest from the server shop using /shop
Place: Place it near farms, mines, or storage areas
Add Items: Put sellable items inside the chest
Profit: Items automatically sell and money is added to your balance

üí° Best Uses

  • Crop Farms: Wheat, carrots, potatoes auto-selling
  • Tree Farms: Wood and saplings processing
  • Mining: Cobblestone and ore disposal
  • Mob Farms: Automatic mob drop selling
‚ö†Ô∏è No Undo: Items are sold immediately and permanently!

Protect your items when you die with graves

ü™¶ Grave Commands - None

üõ°Ô∏è How Graves Work

When you die, a grave is automatically created at your death location containing all your items. Only you can access your graves, keeping your items safe from other players.

‚è∞ Time Limit: Graves last for a limited time (20 m), so make sure to collect your items before they expire!
