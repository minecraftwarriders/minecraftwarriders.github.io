Beginner Guide

Use this path when you first join. It gives players a clear route instead of making them hunt through every command.

1. Join the proxy Add panda.cloudns.nz from Java or Bedrock.
2. Set a home Run /sethome once you find a base location you want to keep.
3. Use the economy Open /shop, check item prices, and use /balance.
Tip: after dying, try /back to return to your previous location.

Core Commands

The commands players use most often are grouped by what they are trying to do.

Server Systems

These are the main systems players will encounter while playing.

EssentialsX Homes, warps, teleport requests, private messages, and economy basics.
AuraSkills Level skills naturally by mining, farming, fighting, fishing, and exploring.
EconomyShopGUI Server shop for buying and selling common resources.
AutoSell Chests Automated selling for farms and resource generation setups.
BigDoors Animated doors, gates, bridges, and large moving entrances.
Social Plugins Marriage, messaging, trading, and other player-to-player tools.

Economy & Trading

Use the shop and price guide before making farms or trading with players.

/shopOpen the main server shop.
/ahOpen the auction house.
/pay [player] [amount]Send money to another player.
/worth [item]Check the server value of an item.

Building & World

War Riders is built around survival plus larger constructions and vehicles. Keep useful commands close while building.

WorldEdit Powerful building tools for staff or permitted builders.
Movecraft Ships, vehicles, and war machines depending on server rules.
Animated Architecture Moving structures like portcullises, bridges, and doors.
